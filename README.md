# TECH TALK AI вЂ” Full Stack Application

An intelligent AI-powered mock interview platform that helps candidates practice technical interviews with real-time emotion detection, resume parsing, AI-generated questions, and detailed performance analytics. Built with FastAPI + React + PostgreSQL + Groq AI.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | FastAPI, Python, PostgreSQL |
| AI Engine | Groq LLM, Tesseract OCR, OpenCV |
| Authentication | Firebase Auth, Google OAuth, JWT tokens |
| Frontend | React.js, Axios |
| Email | SMTP email verification |
| Hosting | Render (backend) + Vercel (frontend) |
| Containerization | Docker |

---

## Features

- **AI Mock Interviews** вЂ” Dynamic questions generated by Groq LLM based on your resume and job role
- **Video Interview Room** вЂ” Real-time video panel with live AI evaluation
- **Emotion Detection** вЂ” Live confidence and emotion analysis during the interview
- **Resume Parser** вЂ” Upload PDF resume and get personalized interview questions
- **Confidence Meter** вЂ” Real-time scoring and feedback during interviews
- **Analytics Dashboard** вЂ” Track performance, scores, and improvement over time
- **Recruiter Dashboard** вЂ” Separate portal for recruiters to review candidate results
- **Detailed Reports** вЂ” Post-interview AI-generated performance reports
- **Authentication** вЂ” Email/password signup with email verification + Google OAuth
- **Candidate Form** вЂ” Collect job role, experience level, and interview preferences

## API Endpoints

| Method | Endpoint | Auth | Description |
|--------|----------|------|-------------|
| POST | /signup | No | Register a new user |
| GET | /verify/{token} | No | Verify email address |
| GET | /check-verified/{email} | No | Check email verification status |
| POST | /login | No | Login with email and password |
| POST | /google-login | No | Login with Google OAuth |
| POST | /parse-resume | Yes | Upload and parse resume |
| GET | /get-profile/{email} | Yes | Get user profile |
| POST | /update-profile | Yes | Update user profile |

## Local Development

### Backend

```bash
cd backend/ai_engine
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### Frontend

```bash
cd frontend
npm install
npm start
```

---

## Environment Variables

### Backend (`backend/ai_engine/.env`)

```
GROQ_API_KEY        = your_groq_api_key
DATABASE_URL        = postgresql://user:password@host/dbname
SECRET_KEY          = your_secret_key
EMAIL_USER          = your_email@gmail.com
EMAIL_PASSWORD      = your_email_app_password
```

### Frontend (`frontend/.env`)

```
REACT_APP_API_URL               = https://tech-talk-ai.onrender.com
REACT_APP_FIREBASE_API_KEY      = your_firebase_api_key
REACT_APP_FIREBASE_AUTH_DOMAIN  = your_project.firebaseapp.com
REACT_APP_FIREBASE_PROJECT_ID   = your_project_id
```

- API keys stored as environment variables, never in code
- Firebase authorized domains restricted to production URLs
