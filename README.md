# TECH TALK AI — Full Stack Application

An intelligent AI-powered mock interview platform that helps candidates practice technical interviews with real-time emotion detection, resume parsing, AI-generated questions, and detailed performance analytics. Built with FastAPI + React + PostgreSQL + Groq AI.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Backend | FastAPI, Python, PostgreSQL |
| AI Engine | Groq LLM, Tesseract OCR, OpenCV |
| Authentication | Firebase Auth, Google OAuth, JWT tokens |
| Frontend | React.js, Axios |
| Email | SMTP email verification |
| Hosting | Render (backend) + Vercel (frontend) |
| Containerization | Docker |

---

## Features

- **AI Mock Interviews** — Dynamic questions generated by Groq LLM based on your resume and job role
- **Video Interview Room** — Real-time video panel with live AI evaluation
- **Emotion Detection** — Live confidence and emotion analysis during the interview
- **Resume Parser** — Upload PDF resume and get personalized interview questions
- **Confidence Meter** — Real-time scoring and feedback during interviews
- **Analytics Dashboard** — Track performance, scores, and improvement over time
- **Recruiter Dashboard** — Separate portal for recruiters to review candidate results
- **Detailed Reports** — Post-interview AI-generated performance reports
- **Authentication** — Email/password signup with email verification + Google OAuth
- **Candidate Form** — Collect job role, experience level, and interview preferences

## API Endpoints

| Method | Endpoint | Auth | Description |
|--------|----------|------|-------------|
| POST | /signup | No | Register a new user |
| GET | /verify/{token} | No | Verify email address |
| GET | /check-verified/{email} | No | Check email verification status |
| POST | /login | No | Login with email and password |
| POST | /google-login | No | Login with Google OAuth |
| POST | /parse-resume | Yes | Upload and parse resume |
| GET | /get-profile/{email} | Yes | Get user profile |
| POST | /update-profile | Yes | Update user profile |

Full interactive docs: https://tech-talk-ai.onrender.com/docs

## Local Development

### Backend

```bash
cd backend/ai_engine
pip install -r requirements.txt
python -m uvicorn main:app --reload 
```

### Frontend

```bash
cd frontend
npm install
npm start
```

## Security

- Passwords hashed with bcrypt
- JWT tokens expire after 24 hours
- Email verification required before login
- API keys stored as environment variables, never in code
- Firebase authorized domains restricted to production URLs
